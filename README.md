# 📊 Sales Performance Dashboard (Excel)

## Overview

This project is an interactive **Sales Performance Dashboard** built using Microsoft Excel.
It provides insights into sales trends, profit margins, and regional performance using dynamic visualisations and filters.

The dashboard helps analyse business performance across different **categories, states, and payment modes**.

## Dashboard Components

### KPI Section

Displays key performance indicators such as:

* Orders
* Total Sales
* Total Profit
* Profit Margin
* Quantity Sold

### Sales Trend

A line chart showing monthly sales performance.

### Profit Margin Analysis

Bar chart comparing profit margins across product sub-categories.

### Sales by State

Bar chart highlighting top-performing states.

### Interactive Filters

* Order Date
* Category
* Payment Mode
* State
